# Requirements
Debian Based
```bash
sudo apt-get install libssl-dev
```
Arch Based
```bash
sudo pacman -S libssl-dev
```
MacOS
```bash
brew install openssl@3
```

# Installation
```bash
make
```

# Usage
```bash
./open-kvm-server # Server
```

```bash
sudo ./open-kvm-client <server_ip> /dev/input/<your kb fd>
```
